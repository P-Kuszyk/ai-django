### Django + AI
Wymagania:  
- python 3.10
- wystawiac API (REST) do frontu
- artykuły (wypełnianie przez AI na podstawie obrazu lub opisu)
- utrzymywanie na GIT (publiczne repo)

### Instalacja
Wykonanie polecenia
```` sh
pip install
````
### Uruchomienie projektu
````sh
python manage.py runserver
````
